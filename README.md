#### 介绍

本项目使用网易云API：https://autumnfish.cn/
本项目部分部分功能为实现，做了隐藏。
本项目使用vue element-ui进行开发，借助Axios技术，所有的请求方法都是封装的network文件夹

#### 安装教程

1.  直接克隆
2.  npm i
3.  npm run serve
